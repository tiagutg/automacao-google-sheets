# Automação de Gestão de Chamados - Google Sheets 🤖

Este projeto demonstra a aplicação de **Google Apps Script (JavaScript)** para automatizar a triagem de chamados técnicos em planilhas.

### 🚀 Funcionalidades
- Varredura automática de linhas para identificar status e prioridade.
- Formatação condicional dinâmica (destaque em vermelho para casos críticos).
- Criação de menu personalizado diretamente na interface do Google Sheets.

### 🧠 Lógica Aplicada
Utilizei conceitos de **matrizes (arrays bidimensionais)** e **estruturas condicionais**, aproveitando minha base em Java e C para manipular o DOM da planilha de forma eficiente.
